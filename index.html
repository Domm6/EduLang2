Start Coding!

  1. Download the edulang.tx file

  2. Download the edulang.py file

  3. Download any dependencies

  4. Put them both in the same folder

  5. Create a test.edulang file in the same folder with a test program

  6. Make sure the file name in this line near the bottom of the edulang.py file mathces with the program file name you want to run: edu_program = edu_mm.model_from_file('test_program.edulang')

  7. Run the program
