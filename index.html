EduLang

A programming language to make studying more simple and effective

More on Github Download

Key Features

  • Support for multiple answers per question
  • Ability to declare variables for later use
  • Set a timer for all quizzes (questions)
  • Keeps track of your session score (correct answers / total attempts)
  • Set an incorrect limit (number of tries per question)
  • Declare hints for each quiz and request them when needed
  • Organize quizzes and definitions by categories (e.g., Math, Science, English)
  • Maintain a running list of definitions for study purposes
  • Include comments in your programs for better organization

Start Coding!

  1. Download the edulang.tx file

  2. Download the edulang.py file

  3. Download any dependencies

  4. Put them both in the same folder

  5. Create a test.edulang file in the same folder with a test program

  6. Make sure the file name in this line near the bottom of the edulang.py file matches with the program file name you want to run: 
    edu_program = edu_mm.model_from_file('test_program.edulang')

  7. Run the program
